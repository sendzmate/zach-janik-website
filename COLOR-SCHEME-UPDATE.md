# Color Scheme Update: Dark Mode with Lime Green

## New Color Palette

### Primary Colors
- **Lime Green**: `#00ff41` - Primary accent color (buttons, links, highlights)
- **Darker Lime**: `#00cc33` - Hover states and interactive elements
- **Pure Black**: `#000000` - Main background
- **Dark Gray**: `#0a0a0a` - Secondary background sections

### Background Colors
- **Dark BG**: `#000000` - Primary background (body, main sections)
- **Darker BG**: `#0a0a0a` - Alternating sections (about, press)
- **Light BG**: `#1a1a1a` - Card backgrounds
- **Gray Shades**:
  - Gray 100: `#1a1a1a` (cards)
  - Gray 200: `#2a2a2a` (borders, hover states)
  - Gray 300: `#3a3a3a` (unused currently)

### Text Colors
- **Primary Text**: `#ffffff` - Headlines, main content
- **Secondary Text**: `#b0b0b0` - Body text, descriptions

### Accent Colors
- **Lime Glow**: Used in shadows and glows (`rgba(0, 255, 65, 0.X)`)

## Shadows & Glows

All shadows now use lime green glows for a cyberpunk/tech aesthetic:
- `rgba(0, 255, 65, 0.1)` - Subtle glow
- `rgba(0, 255, 65, 0.2)` - Medium glow
- `rgba(0, 255, 65, 0.3)` - Strong glow
- `rgba(0, 255, 65, 0.4)` - Intense glow (buttons)

## Typography

Text colors adjusted for dark mode:
- Headlines: Pure white (`#ffffff`)

## Accessibility Notes

The lime green (`#00ff41`) on black backgrounds provides:
- High contrast for readability
- Clear visual hierarchy
- Strong call-to-action visibility
- Modern, tech-forward aesthetic

The white text on black backgrounds ensures:
- Maximum readability
- Reduced eye strain in dark environments
- Professional appearance

## Visual Theme

The dark mode with lime green creates:
- **Cyberpunk/Tech aesthetic** - Modern, futuristic feel
- **High energy** - Bright lime green pops against black

All colors use standard hex and rgba values, ensuring compatibility with:
- All modern browsers
- Mobile devices
- High DPI displays
- Dark mode preferences (native dark mode)
